# Clone do Spotify

Este é um projeto de clone do Spotify, desenvolvido para fins educacionais. O objetivo é criar uma interface semelhante á do Spotify, utilizando tecnologias web modernas.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Tailwind CSS (versão 24.x)
- Node.js (versão 24.x)

## Instalação

1. Clone este repositório:

```bash
git clone https://github.com/Lunnifsp/CloneSpotify
```

2. Navegue até o diretório do projeto: 

```bash
cd CloneSpotify
```

3. Instale as dependências: 

```
npm install
```
